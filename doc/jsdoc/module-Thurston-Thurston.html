Class: Thurston

Thurston~Thurston(geom, options)

Class used to create a scene in the specified geometry

Constructor

new Thurston(geom, options)

Create an instance dedicated to build a scene in the prescribed geometry. Property that are not setup yet, but will be used later are defined as undefined. (Maybe not needed in JS, but good practice I guess.)

Parameters:
Name Type Description
geom Object

a module handing the relevant geometry
options Object

a list of options. See defaultOptions for the list of available options.
