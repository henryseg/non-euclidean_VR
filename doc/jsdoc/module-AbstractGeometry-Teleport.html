AbstractGeometry~Teleport

  1. Home

  3. AbstractGeometry~Teleport

new Teleport( test, isom, inv )

Description

Elementary brick of a discrete subgroups. We describe a discrete subgroups by a set of generators. Each generator is seen as a "teleportation" (to move a point back in the fundamental domain). A Teleport encode both the generator, and the test to decide if a teleportation is needed.

Parameters
Name Type Description
test function

A test saying if a teleportation is needed
isom Isometry

the isometry to apply when teleporting
inv Isometry

the inverse of the isometry (optional) If the inverse is not passed as an argument, it is computed automatically.
