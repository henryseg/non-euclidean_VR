Class: SceneItem

builder~SceneItem(data)

Generic class for items in the scene (objects, lights, etc) This class should never be instantiated directly. Classes that inherit from SceneItem can be instantiated All the properties are not mandatory. Their use will depend on the type of objects. The philosophy is to collect in this class all properties that can be used in more that one inherited class, so that the code is factored as much as possible
